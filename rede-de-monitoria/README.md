# 📚 Controle de Monitoria

Sistema web desenvolvido para auxiliar no gerenciamento e controle de monitorias acadêmicas.

O projeto tem como objetivo centralizar informações importantes sobre monitores, alunos, horários e sessões de monitoria, oferecendo uma experiência simples, intuitiva e organizada para acompanhamento das atividades.

Inspirado na estrutura e organização do projeto referência SIRA, o sistema está sendo desenvolvido com foco em boas práticas de front-end, organização modular do código e experiência do usuário.

---

# 🚀 Objetivos do Projeto

* Facilitar o gerenciamento de monitorias acadêmicas;
* Centralizar informações de alunos e monitores;
* Permitir autenticação de usuários;
* Melhorar a organização das sessões de monitoria;
* Criar uma interface moderna, responsiva e intuitiva;
* Aplicar conceitos de desenvolvimento front-end moderno.

---

* HTML5

* Tailwind CSS

# ✨ Funcionalidades

## 🔐 Sistema de autenticação

* Cadastro de usuários;
* Login de usuários;
* Criptografia de senha utilizando bcryptjs;
* Persistência de sessão;
* Logout do sistema.

## 👤 Gerenciamento de usuários

* Validação de matrícula e e-mail;
* Identificação única utilizando UUID;
* Armazenamento local de dados.

## 🎨 Interface

* Layout moderno e responsivo;
* Dropdown de perfil;
* Feedback visual para interações;
* Organização visual inspirada em plataformas acadêmicas.

# ⚙️ Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/PedroLuisLima/Controle-de-monitoria.git
```

## Acesse a pasta

```bash
cd Controle-de-monitoria
```

## Instale as dependências

```bash
npm install
```

## Execute o projeto

```bash
npm run dev
```
